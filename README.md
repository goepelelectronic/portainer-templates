# Portainer Templates – Göpel electronic GmbH

In diesem Repository werden die Portainer-Templates für die Projekte und Dienste der Firma **Göpel electronic GmbH** gepflegt und bereitgestellt.

This repository contains the Portainer templates for the projects and services of **Göpel electronic GmbH**.

---

## Anleitung: Portainer Template einbinden (Deutsch)

1. Öffnen Sie Ihre Portainer-Oberfläche.
2. Navigieren Sie zu **"App Templates"** im Menü.
3. Klicken Sie auf **"Custom templates"** oder **"Vorlagen verwalten"** (je nach Portainer-Version).
4. Fügen Sie die URL zu dieser `templates.json`-Datei ein:
	```
	https://raw.githubusercontent.com/goepelelectronic/portainer-templates/refs/heads/main/templates.json
	```
5. Speichern Sie die Einstellungen. Die Vorlagen stehen nun zur Verfügung.

---

## Instructions: Add Portainer Template (English)

1. Open your Portainer web interface.
2. Go to **"App Templates"** in the menu.
3. Click on **"Custom templates"** or **"Manage templates"** (depending on your Portainer version).
4. Enter the URL to this `templates.json` file:
	```
	https://raw.githubusercontent.com/goepelelectronic/portainer-templates/refs/heads/main/templates.json
	```
5. Save the settings. The templates are now available for use.
